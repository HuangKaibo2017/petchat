# 更懂它平台个人信息收集清单

**更新日期：** 2026年7月3日
**生效日期：** 2026年7月3日

---

## 引言

更懂它平台（以下简称"我们"或"平台"）深知个人信息对您的重要性。为了向您提供宠物健康评估、情感沟通、电商购物、社交互动等服务，我们需要收集和使用您的部分个人信息。

**本清单将详细说明我们在各业务场景下收集的个人信息类型、收集目的、收集方式以及是否属于必要信息，帮助您清晰了解我们的信息收集情况。** 如对本清单或相关事宜有任何问题，您可以通过网站 www.gengdongta.com 公布的联系方式与我们联系。

---

## 一、 信息收集总览

我们遵循**最小必要原则**，仅在实现特定服务功能所必需的范围内收集您的个人信息。我们将收集的信息分为以下几大类：

1. **账号信息：** 用于注册、登录及账号安全管理
2. **宠物信息：** 用于建立宠物档案及提供宠物相关服务
3. **服务使用信息：** 用于生成AI报告、提供咨询服务
4. **交易信息：** 用于商品购买、订单处理及售后服务
5. **社交信息：** 用于宠友圈互动、评论及内容发布
6. **设备信息：** 用于硬件绑定及IoT服务
7. **日志信息：** 用于服务优化、安全保障及故障排查

根据相关法律法规，以下信息属于敏感个人信息，我们在收集时会特别征得您的同意，并采取更严格的保护措施：

## 五、 信息的存储与保护

### （一）存储期限

我们仅在实现本清单所述目的所需的期限内保留您的个人信息，具体保存期限如下：

| 信息类型 | 保存期限 |
|---------|---------|
| 账号信息 | 账号存续期间及注销后30天 |
| 宠物档案 | 账号存续期间，删除后立即清除 |
| AI报告 | 账号存续期间，用户删除后立即清除 |
| 订单信息 | 交易完成后至少3年（法律法规要求） |
| 社交内容 | 账号存续期间，用户删除后立即清除 |
| 日志信息 | 180天后自动删除 |

### （二）存储地域

您的个人信息将存储在中华人民共和国境内。
